# The Creative Room

## Setup

```bash
npm install
npm run dev
```

## Environment variable

Create a `.env` file:
```
VITE_ANTHROPIC_API_KEY=your_key_here
```

## Deploy to Vercel

1. Push to GitHub
2. Import repo on vercel.com
3. Add environment variable: `VITE_ANTHROPIC_API_KEY` = your Anthropic API key
4. Deploy
